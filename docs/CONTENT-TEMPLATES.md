## Template 1: Consulting Services

Best for: Consultants, Freelancers, Agencies

```

Example:
Name: Sarah Chen
Title: AI Strategy Consultant
Tagline: Helping enterprises unlock 10x productivity with intelligent automation
```

## Content Best Practices

### Hero Section
- Keep tagline under 10 words
- Use action verbs
- Focus on outcomes, not features
- Make CTA buttons clear and specific

### Services/Skills
- Lead with the most valuable offering
- Include specific prices when possible
- List 3-5 features per tier
- Use "Most Popular" on middle tier

### Case Studies
- Always include metrics
- Use before/after format
- Name client type if confidential
- Keep descriptions concise

### Testimonials
- Include name and title
- Focus on results, not process
- Vary the types of clients
- 3 testimonials is ideal

### Contact
- Always include calendar link
- Make email clickable
- Include timezone if relevant
- Less is more - don't overwhelm
