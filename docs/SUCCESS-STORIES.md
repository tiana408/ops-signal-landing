## Featured Stories

### Mark — Stock Trading Signal Application

**Background**: Student in CSE651, new to cloud deployment

**Started With**: This landing page template

**What He Learned**:
- Container-based development workflow
- AI-assisted coding with natural language prompts
- Cloud deployment fundamentals

**What He Built Next**: A stock trading signal application with:
- Backtesting engine to validate trading strategies
- Real-time signal generation
- Performance tracking dashboard

**Where He Deployed**: Vercel

**What's Next**: Exploring how to scale the application and bring it to market

**Key Insight**:
> "The template taught me the fundamentals. Once I understood containers and deployment, I realized I could apply the same patterns to any idea."
